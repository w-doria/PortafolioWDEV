PROYECTOS

Portafolio web con HTML CSS y JavaScript

Uno de mis primeros proyectos en mi camino hacia el desarrollo web fue crear mi propio sitio web para mostrar mi trabajo y habilidades. Utilicé HTML, CSS y JavaScript para diseñar y desarrollar un sitio web personalizado que reflejara mi estilo personal y que fuera fácil de navegar.

Para asegurarme de que mi sitio web fuera fácil de usar y atractivo, utilicé prácticas de diseño responsivo y aseguré que mi sitio fuera accesible desde cualquier dispositivo. También implementé prácticas de SEO, como etiquetas meta descriptivas y descripciones claras, para asegurar que mi sitio web fuera fácilmente descubierto por los motores de búsqueda.

Finalmente, para garantizar que mi sitio web fuera accesible en línea, lo desplegué en la nube utilizando una instancia en Amazon y registré un dominio para facilitar el acceso a mi sitio web.

Estoy muy orgulloso de mi sitio web personal y espero que les guste tanto como a mí. ¡Gracias por visitar mi sitio web y por considerar mi trabajo para cualquier proyecto en el que pueda ser útil!
